# Changelog

All notable changes to PermeantOS will be documented in this file.

## Unreleased

- Added the Agent Memory Graph v0 specification, JSON schema, validation fixture,
  and contract tests for the Phase 1 roadmap item.
- Expanded the v0 schema after reviewing current agent memory systems to cover
  memory tiers, temporal belief metadata, retrieval provenance, session
  checkpoints, compaction summaries, trace spans, handoffs, participants, and
  side-effect approval policy.
- Incorporated Mnemara-inspired local-first memory concepts, including quality
  and historical lifecycle states, trust levels, episodic continuity, salience,
  conflict review, explainable recall planning, score breakdowns, historical
  recall modes, and portable snapshot/package checkpoints.
