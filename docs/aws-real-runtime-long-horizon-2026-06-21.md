## Result

The E2E migration succeeded.

- Source continuation was generated by the local MLX exporter for the exact
  model and prompt.
- The QATQ-compressed KV payload was migrated over production `wss://`/mTLS and
  committed into the AWS vLLM target.
- vLLM generated a 128-token post-migration continuation.
- `source_vs_post_migration` matched exactly at 16, 32, 64, and 128 tokens.
- `baseline_vs_post_migration` matched exactly at 16, 32, 64, and 128 tokens.
- `max_complete_exact_horizon` was 128.
- Reverse vLLM runtime export succeeded, then the origin MLX exporter imported
  the target-advanced boundary and generated a new 128-token origin
  continuation.
- Target-side Agent Memory Graph resume executed pending work and wrote a new
  artifact on AWS.
- Origin return-home verification consumed the AWS-updated graph/artifact proof
  and wrote a new origin-side continuation artifact.
- Cleanup completed and a direct AWS verification confirmed the instance was
  terminated and the temporary security group/key pair no longer existed.
