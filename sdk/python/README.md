# PermeantOS Python SDK

The PermeantOS Python SDK provides a small client and Agent Memory Graph helper
surface for local migration demos and Python-native runtime adapter work.

This package is pre-publication. It is intentionally marked as unpublished in
`pyproject.toml` until the repository has the documented real-release path for
Python package ownership, credentials, package validation, and versioned
publishing.

## Local Smoke Test

```bash
python sdk/python/test_sdk.py
```

## Public Versions

The SDK exports the current public exchange/schema identifiers:

- `USXF_VERSION`
- `AGENT_MEMORY_GRAPH_SCHEMA_ID`
- `AGENT_MEMORY_GRAPH_GRAPH_VERSION`
